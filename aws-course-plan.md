
## AWS Course Curriculum 

### Course Overview

This AWS course is designed to provide **end-to-end practical knowledge of Amazon Web Services**, covering cloud fundamentals, core AWS services, architecture, security, automation, monitoring, and real-world projects. The program focuses heavily on **hands-on labs, real-time scenarios, and certification readiness**. 

---

## Module-wise Summary

### 1. Cloud Computing Fundamentals

* Evolution and history of cloud computing
* Distributed computing concepts and challenges
* Client-server architecture
* Benefits and advantages of cloud computing
* Why cloud computing is essential for modern IT

---

### 2. Cloud Deployment Models

* Private Cloud
* Public Cloud
* Hybrid Cloud
* Use cases and real-world examples for each model

---

### 3. Cloud Service Models

* Software as a Service (SaaS)
* Platform as a Service (PaaS)
* Infrastructure as a Service (IaaS)
* Comparison and practical relevance

---

### 4. Linux Basics

* Essential Linux commands
* Basic Linux administration
* Preparing Linux environments for AWS workloads

---

### 5. Introduction to AWS & Global Infrastructure

* AWS global infrastructure overview
* Regions, Availability Zones, and Edge Locations
* AWS Management Console, CLI, and SDK
* Creating and managing AWS Free Tier accounts

---

### 6. Core AWS Services Overview

Introduction to major AWS services including:

* Compute, storage, networking, databases
* Messaging, monitoring, security, and automation services
* Containers and serverless basics

---

## Compute & Storage Services

### 7. Amazon EC2 (Elastic Compute Cloud)

* EC2 instance types and pricing models
* AMIs, key pairs, security groups, tags
* Elastic IPs, instance roles, placement groups
* Instance metadata and bootstrapping
* Hands-on: Linux & Windows EC2, web servers, AMIs

---

### 8. Amazon EBS (Elastic Block Store)

* Volume types and performance
* Encryption and snapshots
* Attaching, detaching, resizing volumes
* Cross-region snapshot copy
* Hands-on disk management (Linux & Windows)

---

### 9. Amazon EFS (Elastic File System)

* File system concepts and use cases
* Mount targets and security
* Monitoring and data protection
* Hands-on EFS creation and mounting

---

### 10. Amazon S3 (Simple Storage Service)

* Buckets, objects, and storage classes
* Versioning, lifecycle policies, and encryption
* Access control (ACLs, bucket policies)
* Static website hosting
* Cross-region replication
* Hands-on via CLI and Python

---

### 11. Amazon Glacier

* Archival storage concepts
* Vaults, archives, and retrieval policies
* Backup strategies and lifecycle integration
* Hands-on Glacier operations

---
